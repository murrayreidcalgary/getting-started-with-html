HTML Elements

An element usually consists of:
<opening tag> content </closing tag>
In some cases and element may be a "void tag", where there is no content, nor a closing tag.

Elements may be inline or block oriented.

Tags consist of a mnemonic surrounded by a leading "less than" symbol and followed by a "greater than" symbol. The closing tag has a "forward slash" symbol preceding the mnemonic. Some opening tags have Atrributes that modify their behaviour - these appear after the mnemonic and before the closing "greater than" symbol.

In the following list, the menemonic of the tag is given, followed by an indication of whether it is inline or block oriented in partentheses. This is followed with a short description of what the tag does. Where Attribute(s) are known they are listed on succeeding lines.

  • b (i) Bold
  • em (i) EMphasis, italics
  • a (i) Anchor; link to other stuff
    • href="url" where to go to
  • img (i) IMaGe; include a picture here
    • src="ref" where to find the picture
    • alt="text" description of the picture
  • br (i) BReak; start a new line; [VOID Tag]
  • p (b) Paragraph
  • div (b) create a block of stuff, a DIVision
  • span (i) create a block of stuff that SPANs from here to the closing tag
  • strong (i) provide STRONG emphasis to text
  • form (b) create a FORM
  • head (b) create a HEADing for the document
  • body (b) create the BODY of the document
  • html (b) the defifinition of the entire HTML document
  • h1...h6 (i) Heading levels 1...6
  • ol (i?) Ordered List
